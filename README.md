# 水喝千束的日志\n\n一个支持 Markdown、卡片科技风的静态博客，可直接部署到 GitHub Pages。\n\n## 部署方法\n1. 上传至 GitHub 仓库（如 `waterchisato-blog`）。\n2. 在仓库设置中启用 Pages（Source: main /root）。\n3. 访问 `https://YOUR_USERNAME.github.io/YOUR_REPO/`。\n\n## 目录结构\n````\nwaterchisato-blog/\n├── index.html\n├── post.html\n├── posts/\n│   ├── posts.json\n│   ├── first-post.md\n│   └── example.md\n├── scripts/\n│   └── main.js\n├── styles/\n│   └── style.css\n└── assets/\n    └── logo.png\n````
